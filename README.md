# SPA-curriculum

SPA MERN básica con un backend Express y un frontend React (Vite).

## Estructura

- `backend/`: API Express + Mongoose.
- `frontend/`: SPA React con MUI, React Router, axios y SimpleBar.

## Uso rápido

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
